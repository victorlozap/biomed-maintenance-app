# BioMed HUSJ - Sistema de Mantenimiento Biomédico

Aplicación para la gestión y seguimiento del mantenimiento preventivo y correctivo de equipos biomédicos en el Hospital Universitario San Jorge.

## 🚀 Inicio Rápido
1. Abre una terminal en la carpeta `biomed-maintenance-app`.
2. Instala dependencias: `npm install`.
3. Inicia el servidor de desarrollo: `npm run dev`.

## 📁 Estructura
- `/biomed-maintenance-app`: Aplicación React + Vite.
- `/docs`: Documentación del proyecto.
- `/formatos`: Archivos base y plantillas.
- `/scripts`: Scripts de procesamiento de datos.

## 🛠️ Tecnologías
- Vite / React / TypeScript
- Vanilla CSS
- Python (para extracción de datos)
