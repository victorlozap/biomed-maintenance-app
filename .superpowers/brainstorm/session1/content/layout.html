¿Cómo quieres organizar la navegación principal?

Elige el layout general de la app de mantenimiento biomédico

A — Sidebar lateral fijo

Menú vertical siempre visible. Ideal para apps con muchas secciones. Navegación rápida sin perder contexto.

B — Barra superior + tabs

Navegación horizontal en la parte superior. Más espacio vertical para tablas y formularios grandes.

C — Header fijo + sidebar expansible

Header con logo y perfil arriba. Sidebar colapsable para ganar espacio. El más profesional y flexible.
