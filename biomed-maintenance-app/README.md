# 🏥 BioMed HUSJ - Sistema de Gestión de Mantenimiento

Sistema avanzado para la gestión de mantenimiento biomédico e inventario clínico del **Hospital Universitario San Jorge**, desarrollado como proyecto final para la Tecnología en Análisis y Desarrollo de Software.

## 🌟 Características Principales
- **Inventario Inteligente**: Control total sobre 431 activos con búsqueda dinámica y trazabilidad.
- **Mantenimiento Correctivo**: Generación de reportes institucionales (GRF3MAN-FR134) en PDF con firmas dinámicas.
- **Mantenimiento Preventivo**: Flujos de trabajo guiados para protocolos de equipos de alta complejidad.
- **Dashboard de Control**: Visualización en tiempo real del estado de los equipos y rendimiento técnico.

## 🛠 Instalación y Desarrollo
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`
3. Configurar variables de entorno en `.env` (Supabase URL/Key).
4. Ejecutar en local: `npm run dev`
5. Compilar para producción: `npm run build`
